# Verewaltung
Static multi-page landing site (HTML/CSS/JS).

## Pages

- `index.html` (Startseite)
- `ueber-uns.html`
- `referenzen.html`
- `blog.html`
- `kontakt.html`

## Local preview

Any static server works. Example:

```bash
python3 -m http.server 5173
```

Then open `http://localhost:5173`.

## Images

Add real photos into `assets/` (see `assets/README.md`).
